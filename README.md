一个轻量便捷的投影查看器

- `导入` 导入投影文件
- `导出` 导出投影数据 (文本&表格)
- `分类导出` 导出分类投影数据 (文本&表格)
- `简洁分析` 轻量分析，只会显示方块名与数量
- `容器分析` 容器分析，显示容器中存储的方块和基础信息
- `生成图形投影` 快捷生成一个常规立方体
- `替换特定方块` 快速替换/限制投影里的不同方块
- `3D渲染` 3D渲染目标投影 (可能引起卡顿)
- `跨版本转换投影` 将默认最新版本投影转换到1.16/1.13版本的投影文件
- `界面颜色` 更改界面主题色 (蔚蓝色,亮绿色,暗灰色,深蓝色,粉色)
